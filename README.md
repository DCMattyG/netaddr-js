# netaddr-js

JavaScript equivalent to Python `netaddr`, with precision-safe IP math
(`BigInt`) and familiar APIs for IP/CIDR/range/set/EUI workflows.

## Attribution

`netaddr-js` is explicitly inspired by the Python
[`netaddr`](https://github.com/netaddr/netaddr) project.

Credit belongs to the Python `netaddr` maintainers and contributors for the
original project and ecosystem impact. `netaddr-js` is an independent
JavaScript implementation and is not an official Python `netaddr` port.
